# CI/CDefense

*STILL IN ACTIVE DEVELOPMENT*

**Practice software supply chain security with CI/CDefense!**

Build your own Terraform lab for CI/CD pipeline and software supply chain security testing.

## Structure

- `azure/` — Azure deployment (Windows Server 2022, Ubuntu, Kali)
- `gcp/` — GCP deployment

## Usage

```powershell
cd azure
terraform init
terraform plan
terraform apply
terraform destroy
```
